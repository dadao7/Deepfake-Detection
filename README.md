# Deepfake-Detection

## 프로젝트 소개

본 프로젝트는 딥러닝 기반의 딥페이크 탐지 시스템이다.
M2F2-Det 기반 탐지 모델에 Heatmap 기반 Top-K Crop, 국소패치 정규화, 동적 결합 네트워크를 적용하여 미세 위조 흔적 탐지 성능을 향상시켰다. 
또한 SIDA 기반 VLM/LLM 설명 생성 모듈을 활용하여 단순한 REAL/FAKE 분류 결과뿐 아니라 탐지 근거를 자연어로 제공하는 설명 가능한 AI 시스템을 구현하였다.

## 프로젝트 목적

- 딥페이크 이미지로 인한 허위 정보 및 디지털 범죄 문제에 대응
- 사람의 육안으로 구분하기 어려운 위조 이미지를 자동 탐지
- 전역 특징(Global Feature)과 국소 특징(Local Feature)을 함께 활용하여 탐지 성능 향상
- 단순 결과만 제공하는 것이 아닌 AI가 어떤 특징을 근거로 판단했는지 설명 제공
- 모바일 환경에서 누구나 손쉽게 딥페이크 여부를 검증할 수 있는 서비스 제공

## 주요 기술

### M2F2-Det 기반 특징 추출

M2F2-Det(Multi-scale Multi-modal Feature Fusion Detector)의 구조를 기반으로 이미지의 전역 특징과 세부 영역 특징을 추출한다. 얼굴 전체 정보뿐만 아니라 눈, 피부 질감, 경계 영역 등 국소적인 위조 흔적을 함께 분석하여 탐지 성능을 향상시킨다.

### SIDA 기반 설명 생성

SIDA(Synthetic Image Detection Assistant)를 활용하여 탐지 결과에 대한 자연어 설명을 생성한다. 단순히 REAL 또는 FAKE 결과만 제공하는 것이 아니라 이미지의 어떤 특징이 위조 흔적으로 판단되었는지 사용자에게 설명 가능한 AI(XAI) 형태로 제공한다.

### 모바일 애플리케이션

React Native(Expo) 기반으로 구현하여 사용자가 이미지를 업로드하고 분석 결과와 AI 리포트를 직관적으로 확인할 수 있도록 설계하였다.

## 기대 효과

- 딥페이크 이미지에 대한 사용자 경각심 향상
- 온라인상의 허위 정보 확산 방지
- 디지털 콘텐츠 신뢰성 검증 지원
- 설명 가능한 AI 기반 탐지 시스템 제공
- 향후 영상 단위 딥페이크 탐지 및 실시간 분석 기능으로 확장 가능
